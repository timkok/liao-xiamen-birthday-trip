# 厦门 70 岁生日家庭行程

为六位家人准备的厦门 6 天 5 晚生日旅行 Web App，包含老人看图版行程、菜单预算、出发清单、高德导航、酒店沟通模板和旅途画卷。

## 在线访问

- GitHub Pages：<https://timkok.github.io/liao-xiamen-birthday-trip/>
- Cloudflare Pages：<https://liao-xiamen-birthday-trip.pages.dev/>

## 本地运行

```bash
npm install
npm run dev
```

推送到 `main` 分支后，GitHub Actions 会自动更新 GitHub Pages。
